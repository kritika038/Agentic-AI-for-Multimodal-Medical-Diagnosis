Features

Designed to demonstrate how specialized AI agents can collaborate inside a healthcare workflow.

Imaging diagnosis

Analyzes chest X-ray input to estimate pneumonia presence and surface supporting evidence for downstream reasoning.

Clinical risk estimation

Uses structured patient measurements to estimate diabetes risk and complement the imaging branch.

Agentic orchestration

Routes evidence across intake, diagnosis, localization, severity, uncertainty, doctor, and report agents.

Workflow

How the live system moves from input data to triage-oriented output.

01

Collect multimodal inputs

The workflow accepts a chest X-ray image together with structured clinical measurements and patient context.

02

Run specialized agents

Independent agents evaluate diagnosis, localization, severity, clinical risk, uncertainty, and evidence fusion.

03

Generate triage guidance

The system returns an explainable report with multimodal reasoning, confidence context, and triage-focused recommendations.

It demonstrates a multimodal pipeline that blends chest X-ray reasoning with tabular health-risk assessment and triage reporting.
